# evandroripka.dev

Interactive portfolio website built with Vite, Three.js, and GSAP.

## Goals
- Cinematic hero section with a dynamic environment (day/night + weather)
- High performance and clean architecture
- Expandable scene system (swap placeholder geometry for 3D assets later)

## Tech Stack
- Vite
- Three.js
- GSAP

## Getting Started
```bash
npm install
npm run dev
